# 🏡 Airbnb Clone (Full Stack Project)

A full-stack Airbnb clone built using Node.js, Express, MongoDB, and EJS. Users can explore listings, create their own properties, and view locations on an interactive map.

---

## 🌐 Live Demo

👉 https://airbnb-project-42k3.onrender.com/listings

Explore real-time listings, create new properties, and view locations on the map.


## 🚀 Features

- 🏠 Create, edit, and delete listings
- 🔐 User authentication (Login / Signup)
- 🗺️ Interactive map using Mapbox
- 💬 Flash messages for user feedback
- 💾 MongoDB database integration
- 📷 Image support via URLs
- 📍 Geolocation support (coordinates stored in DB)

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js  
- **Frontend:** EJS, Bootstrap  
- **Database:** MongoDB Atlas  
- **Map:** Mapbox  
- **Authentication:** Passport.js  

---


## 🌍 API / Routes

- `/listings` → Show all listings  
- `/listings/new` → Create listing  
- `/listings/:id` → View single listing  
- `/login` → User login  
- `/signup` → User registration
